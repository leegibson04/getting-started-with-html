How the Web Works

1.0 World Wide Web: A collection of HTML documents
2.0 HTML: HyperText Markup Language, the basis for every webpage, main type of document on the web
3.0 Hyper Links: Make the web - web like

Type of documents can be found on the web: Plain text, HTML, Images, Videos, Music

Major Pieces of the Web

1.0 Browser: A program that runs on your computer to display files found on the web (IE, Chrome, Firefox)
2.0 The Internet: the worlds largest computer network
3.0 Servers: computers that host the files the make up the web
4.0 HTTP: the main protocol of the web

The browser uses the internet to make request of servers through HTTP. The servers respond with files which the browser displays.

HTML

1.0 Text content: what you see
2.0 Markup: what it looks like
3.0 References to other documents: images or videos
4.0 Links to other pages
5.0 Uses tags to form elements
Computers are stupid: sensitive to typos or small mistakes (like a missing end tag)
